Primeira etapa: Configurando o Serviço do SNMP do pfSense
Services 
	SNMP
		SNMP Daemon
			Enable: ON
		SNMP Daemon Settings
			Polling Port: 161 (Default)
			System Location: Bora para Pratica 
			System Contact: Robson Vaamonde 
			Read Community String: vaamonde
		SNMP Traps Enable
			Enable: OFF (Default)
		SNMP Modules
			Mibil: ON
			Netgraph: ON
			PF: ON
			Host Resources: ON
			UCD: ON
			Regex: ON
		Interface Binding
			Bind Interfaces: LAN
	<Save>

Segunda etapa: Testando o SNMP do pfSense nos Desktop GNU/Linux e Microsoft Windows

Linux Mint Terminal: Ctrl+Alt+T
	sudo apt install snmp snmp-mibs-downloader
