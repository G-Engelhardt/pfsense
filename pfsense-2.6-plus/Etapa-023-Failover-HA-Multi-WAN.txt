Observações das configurações utilizadas nessa documentação
(Default) = Configuração padrão do pfSense = ON ou OFF ou Value
(Disable) = Desabilitado nessa configuração = OFF
(Enable)  = Habilitado nessa configuração = ON
(Remove)  = Removido dessa configuração = Sem valor ou opção
<***>     = Botão de confirmação ou de aplicar as configurações

Primeira etapa: Configuração do Monitoramento dos Gateways no pfSense
System
	Advanced
		Gateway Monitoring
			State Killing on Gateway Failure: ON (Default)
			Skip rules when gateway is down: ON (Default)
	<Save>

Segunda etapa: Configuração do Grupo de Gateways de Failover (Falha) e Failback (Retorno da Falha) no pfSense
System
	Routing
		Gateway Groups
			+ADD
				Edit Gateway Group Entry
					Group Name: HA_WAN
					Gateway Priority
						WAN_VIVO_DHCP  Tier 1  Interface Address  Interface WAN_VIVO_DHCP
						WAN_NET_DHCP   Tier 2  Interface Address  Interface WAN_NET_DHCP
					Trigger Level: Packet Loss or High Latency (Enable)
					Description: Grupo de Gateways HA das Interfaces WAN Vivo e Net
				<Save>
			<Apply Changes>

Terceira etapa: Alterando o Gateway padrão do pfSense
System
	Routing
		Gateways
			Default Gateways
				Default gateway IPv4: HA_WAN (Grupo de Gateways HA das Interfaces WAN Vivo e Net) (Enable)
			<Save>
		<Apply Changes>
